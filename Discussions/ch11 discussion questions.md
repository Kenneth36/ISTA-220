2.  How do you call a method that takes an arbitrary number of arguments?

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;**Answer:**  Example (1, 2, 3, 4, 5) or Example (alpha, bravo, charlie, delta)

3. Why can’t you use an array to pass an arbitrary number of arguments to a method? 

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;**Answer:**  The size is a fixed limit in an array.  You have to declare the limit once you create it

4.  How many parameters can a method have?

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;**Answer:**   Any number of parameters.  In other words, a variable number of parameters

5.  Do parameter arguments have to have the same type?  

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;**Answer:**  Yes, you have to have the same type

6.  What is the difference between a method that takes a parameter argument and one that takes optional arguments?

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;**Answer:**  You have to specify the parameters by name if you use an arbitrary argument.  With an optional argument you have a limited number.

7.  How do you define a method that takes different (and arbitrary) types of arguments?

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;**Answer:**  By using your object.
