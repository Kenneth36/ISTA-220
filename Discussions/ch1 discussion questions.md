# Chapter 1.  Discussion Questions 
## Kenneth Clark
### August 21, 2017

1.  What is a console app?

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;**Answer:**   A console application (console app) is an application that runs in a Command Prompt window instead of providing a graphical user interface (GUI).

2.  What does Main() (the main method) do in a console application?

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;**Answer:**   The Main method designates the program’s (console application) entry point.

3.  What is the purpose of a namespace?

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;**Answer:**   A namespace helps to solve the problem with name-clashing as the number of names increases within a program. (Think of a kitchen drawer or a table)..

4.  What is an assembly?

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;**Answer:**   An assembly is a file that usually has the .dll file name extension. (the project) (the executable where everything is tied together)

5.  What is the starting point in a graphical application?

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;**Answer:**   The starting point in a graphical application is the Code and Text Editor window.

6.  What does Build do?

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;**Answer:**   Build verifies that the project builds successfully. (compiles C# code so that it can run)

7.  What does Debug do? 

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;**Answer:**   Debug gives a pair of numbers that track the frame rate which allows for developers to determine when an application starts to become less responsive than it should be.  (Runs the program line by line)
